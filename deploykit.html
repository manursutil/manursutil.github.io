DeployKit

Herramienta CLI para desplegar frontend y backend de forma independiente por tenant sobre infraestructura AWS existente, con despliegues fijados a commits inmutables.

  • TypeScript
  • AWS
  • Docker
  • Node.js
Ver repositorio ↗
01 · Seguro

SSM, no SSH

Los despliegues de backend se ejecutan mediante AWS SSM y las referencias se resuelven a un commit.

02 · Independiente

Por tenant

Frontend y backend se despliegan, conservan y revierten de forma independiente para cada tenant.

03 · Reproducible

Simulación local

Un entorno Docker permite recorrer el flujo completo sin depender de una cuenta de AWS.
