Snake AI

Un agente que aprende a jugar a Snake mediante una red DQN. El proyecto conecta un motor rápido escrito en C y renderizado con raylib con un agente en Python y PyTorch, separando juego, entrenamiento y visualización.

  • Python
  • C
  • PyTorch
  • raylib

02 / 03

DevOps · Producto

DeployKit

Herramienta CLI para desplegar frontend y backend de forma independiente por tenant sobre infraestructura AWS existente. Fija cada despliegue a un commit, usa SSM en lugar de SSH e incorpora historial, bloqueos, rollback y simulación local.

  • TypeScript
  • AWS
  • Docker
  • Node.js
Ver repositorio ↗
03 / 03

Datos · Producto local

Pulso Lanzarote

Una guía viva de eventos de la isla. Recopila agendas locales, normaliza y elimina duplicados, y selecciona planes relevantes para publicarlos en web, Telegram y un calendario iCalendar, con enriquecimiento semántico local mediante Ollama.

  • Python
  • FastAPI
  • SQLite
  • Ollama
